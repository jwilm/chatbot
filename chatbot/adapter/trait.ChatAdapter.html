Chatbot is extensible in both message sources and command handling. To add a new message source, create a type that implements the ChatAdapter trait.

Required Methods

fn get_name(&self) -> &str

The name of the adapter which is used internally as a map key and for debugging.

fn process_events(&self, &Chatbot, Sender<IncomingMessage>)

ChatAdapters must implement process_events. What this method does will vary wildly by adapter. At the very least, it must generate IncominMessages from its input, send them via the Sender that's passed in. The main loop has the other end of this receiver. The IncomingMessage must be constructed with a Sender<OutgoingMessage> for which the adapter listens on the Receiver to send messages back to the service.
